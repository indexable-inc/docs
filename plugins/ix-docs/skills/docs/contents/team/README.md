> [!IMPORTANT]

Support response is great today because the founder and the rest of the team answer directly; scaling that is an active priority. Single-founder company risk is real; if it's unacceptable for your use case, wait or pick someone else. Business-continuity plan in [reliability](../reliability.md#business-continuity).
